# MusicRecommendationSystem
## 基于机器学习——音乐推荐系统  
### 系统介绍  
    搭建Web端音乐网站作为收集数据的工具。通过Spark ALS算法（根据用户播放过很多相同的歌曲，来判断他们喜欢同一个艺术家的歌曲最终给用户推荐适合他的艺术家的作品）给用户推荐歌曲。
